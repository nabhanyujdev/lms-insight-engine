Score derivation

How the learning scores are built.

Activity Score = 0.35 x Login Frequency + 0.35 x Time Spent Modules + 0.30 x Resource Access Frequency
Participation Score = 0.60 x Participation Forums + 0.40 x Session Duration Average
Performance Score = 0.65 x Quiz Performance Average + 0.35 x Assignment Submissions
